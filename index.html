What is D.E.A.R. M.A.N.?

D.E.A.R. M.A.N. is a skill coming from Dialectical Behavioral Therapy, or DBT. It is used to have a more successful interaction with someone when asking for something or saying no to something. D.E.A.R. M.A.N. is an acronym used for remembering the steps in this process. It breaks down into:

  1. Describe
  2. Express
  3. Assert
  4. Reinforce
  5. Mindful
  6. Appear Confident
  7. Negotiate

These all break down into a script one can follow for more successful interactions. Each part of D.E.A.R. M.A.N. is explained below and used to create a script to practice before an upcoming interaction.
